## Como publicar

1. Extraia o ZIP.
2. Envie o conteúdo da pasta `wa-site-final-custo-zero` para a pasta pública da hospedagem:
   - public_html
   - www
   - htdocs
   - ou raiz do domínio.
3. Acesse o domínio.

## Ajuste antes de publicar

Troque `https://seudominio.com.br/` pelo domínio real em:

## Avaliações do Google

A integração automática com avaliações do Google foi removida nesta versão para manter custo zero.

Recomendação:
- Usar botão/link para a ficha do Google Maps.
- No futuro, se quiser avaliações automáticas, usar Google Places API com cache de 3 dias, limite de cota e API Key restrita.
